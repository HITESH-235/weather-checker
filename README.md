# Weather Checker

A simple weather application built with **React + Vite** that allows users to search for weather conditions in different cities using the OpenWeather API.

## Features

* Search weather by city
* Display temperature and weather condition
* Show humidity information
* Toggle between Celsius and Fahrenheit
* Recent search history stored using localStorage
* Clean responsive interface

## Tech Stack

* React
* Vite
* JavaScript
* CSS
* OpenWeather API

## Live Demo

https://HITESH-235.github.io/weather-checker

## Installation

Clone the repository:

git clone https://github.com/HITESH-235/weather-checker.git
cd weather-checker

Install dependencies:

npm install

Run the development server:

npm run dev

## Build

To build the project for production:

npm run build

## Deployment

The project is deployed using **GitHub Pages**.
